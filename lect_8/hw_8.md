Задача 1
--------
Для мультиплитивной группы поля $GF(5^2)$, считая $x$ - примитивным элементом, определить порядки всех элементов поля.

Мультипликативная группа $GF(5^2)*$ имеет порядок $25 - 1 = 24$.

По условию, $x$ - образующий элемент мультипликативной группы. Тогда:
$$x^{24} = 1$$

ord(xᵏ) = 24 / gcd(24, k)

## Итоговое распределение порядков
- Порядок 1: {x⁰ = 1}
- Порядок 2: {x¹²}
- Порядок 3: {x⁸, x¹⁶}
- Порядок 4: {x⁶, x¹⁸}
- Порядок 6: {x⁴, x²⁰}
- Порядок 8: {x³, x⁹, x¹⁵, x²¹}
- Порядок 12: {x², x¹⁰, x¹⁴, x²²}
- Порядок 24: {x¹, x⁵, x⁷, x¹¹, x¹³, x¹⁷, x¹⁹, x²³}

Всего 24 элемента, каждый ненулевой элемент поля входит ровно в один класс.
