Задача 1
--------
Для БЧХ-кода $(15,7)$ с минимальным расстоянием $5$
- построить порождающую матрицу в двоичном виде
---
Рассматривается двоичный примитивный БЧХ-код с длиной n = 15 = 2^4 - 1

Выберем примитивный многочлен степени 4:

p(x) = x⁴ + x + 1

Тогда:
α⁴ = α + 1

Это позволяет выразить все степени α через базис:
{1, α, α², α³}

---
Так как код двоичный, используем классы по умножению на 2:

### Класс C₁:
1 → 2 → 4 → 8 → 16 ≡ 1 (mod 15)

C₁ = {1, 2, 4, 8}

### Класс C₃:
3 → 6 → 12 → 24 ≡ 9 → 18 ≡ 3

C₃ = {3, 6, 12, 9}

---

Каждому классу соответствует один минимальный многочлен.

### Для класса C₁ = {1,2,4,8}

Минимальный многочлен:
M₁(x) = (x − α)(x − α²)(x − α⁴)(x − α⁸)
M₁(x) = x⁴ + x + 1

---

### Для класса C₃ = {3,6,9,12}

Минимальный многочлен:
M₃(x) = (x − α³)(x − α⁶)(x − α⁹)(x − α¹²)
M₃(x) = x⁴ + x³ + x² + x + 1

---

g(x) = M₁(x) · M₃(x)

M₁(x) · M₃(x) = (x⁴ + x + 1)(x⁴ + x³ + x² + x + 1) =

= x⁸ + x⁷ + x⁶ + x⁵ + x⁴  
+ x⁵ + x⁴ + x³ + x² + x  
+ x⁴ + x³ + x² + x + 1  

Теперь складываем по mod 2:

- x⁸ → 1  
- x⁷ → 1  
- x⁶ → 1  
- x⁵ → 1+1 = 0  
- x⁴ → 1+1+1 = 1  
- x³ → 1+1 = 0  
- x² → 1+1 = 0  
- x → 1+1 = 0  
- 1 → 1  

Итоговый многочлен:

g(x) = x⁸ + x⁷ + x⁶ + x⁴ + 1

---

Запишем коэффициенты g(x) от старшей степени к младшей:

g(x) = x⁸ + x⁷ + x⁶ + 0·x⁵ + x⁴ + 0·x³ + 0·x² + 0·x + 1

Вектор:
(1 1 1 0 1 0 0 0 1)

Длина вектора = 9
